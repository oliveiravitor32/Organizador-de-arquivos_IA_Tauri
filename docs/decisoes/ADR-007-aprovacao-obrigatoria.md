# ADR-007: Aprovação Obrigatória para Alterações

Status: Aceito

## Contexto

O sistema possui capacidade de sugerir reorganizações físicas de arquivos.

Alterações automáticas representam risco elevado.

## Decisão

Nenhuma alteração física será aplicada sem aprovação explícita do usuário.

## Justificativa

- Segurança.
- Transparência.
- Confiabilidade.

## Consequências

Positivas:

- Menor risco de perda de dados.
- Maior confiança do usuário.

Negativas:

- Fluxo menos automatizado.
